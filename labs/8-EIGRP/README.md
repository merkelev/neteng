### EIGRP

**Цель: Настроить EIGRP в С.-Петербург. Использовать named EIGRP.**  
1. R32 получает только маршрут по-умолчанию  
2. R16-17 анонсируют только суммарные префиксы

|        | Et0/3     | FE80::C:2 |
